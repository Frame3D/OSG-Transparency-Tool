---------------------
FORMAT OF INPUT FILE
---------------------
line 1: crd file
line 2: ebc file
line 3: singe-sided (1) or double-sided(0) flag
line 4: transparency level
line 5: r g b color
If multiple ebc files are needed, add them by repeating from line2 to line 5.
----------------------
NOTICE
----------------------
1. The order from line 2 to line 5 can not be changed.
2. # indicates the start of comments, comments could only be added at the end of each line.
